Welcome to cfsprinter's documentation!
======================================

``pagerprinter`` is a program to perform actions based on paged messages generated by the South Australian Country Fire Service.

This application is not supported, endorsed, or created by any Fire Service.  While care has been taken while writing this application, it is created for informational purposes only and should not be relied upon in the event of an emergency.
